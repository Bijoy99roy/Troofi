# NFT Marketplace

**Project Name:** Troofi

**Value Proposition:**

Troofi helps creators and collectors to list, buy and trade NFTs in a decentralized manner. Troofi is built on solana making each operations cheap and fast. Creators get direct control over their assets and buyers benefit from a fair and tamperproof marketplace.

**Key Differenciator:**

While the NFT marketplace is already well-established, Troofi introduces key features that makes it stand out:

**1. Fair Discovery**

We don't use any form of algorithm for ranking. This ensures that all project have equal visibility. Our marketplace uses onchain data to for discovering assets, preventing bias or manipulation in what user sees.

**2. Simplified User Experience**

Troofi is build with it's users in mind and that makes an intuitive, seamless experience for both new and experienced NFT traders.

**3. Low Cost and Fast Transactions**

Leveraging Solana's high speed and low cost transactions, Troofi provides affordable and scalable trading. Making it ideal for indie creator and collectors alike.

## Target User Profile

- **NFT Creator:** Wants to list and sell NFTs permissionlessly. values fair exposue and fair trade.

- **NFT Collector:**  Browses NFTs, buys assets and wants guarentees that listings and pricing are transaprarent and tamperproof

- **Beginner Web3 User:** New to NFTs, want an easy, safe and intuitive experience. Values simple onboarding, easy wallet connections and clear steps to buy their first NFT.

- **NFT Flipper/Trader:** Actively buys and sells NFTs for profit. Needs fast execution and fees low enough to make multiple trades cost effective.


## User Story ID: TF-001

**1. User Persona**

- **Name:** Daniel
- **Role:** NFT creator
- **Goal:** List his NFTs for sale and get fair visibility

**2. User Story**

I have my own NFTs and I want to list them for sale so I can reach collectors worldwide and sell NFTs to them.

**3 Acceptance Criteris**

**Functionality**

- The platform allows the creator to list his NFTs for sale and set his own price
- The platform allows the creator to remove his listing


**User Interaction**

- The user connect their wallet to view all the NFTs they already hold
- The user selects an NFT and sets a sale price
- The user signs a single transaction to list the NFT

**Security**

- Only NFTs actually owned by the user can be listed
